# Project Governance

## 1. Overview

The Supreme AI Swarm Orchestrator (SASO) is a proprietary open-source project owned and managed by **RJ Business Solutions**. Project governance ensures transparent, consistent decision-making, software supply chain security, and organizational stability for enterprise clients and government agencies.

---

## 2. Roles and Responsibilities

### 👑 Project Sponsor
*   **Role**: **Rick Jefferson** (Owner, RJ Business Solutions)
*   **Authority**: Holds absolute veto power over all technical, business, and licensing choices. Responsible for funding, licensing policies, and corporate strategic direction.

### 🛠️ Maintainers
*   **Role**: Developers authorized by the Project Sponsor to review, approve, and merge pull requests.
*   **Authority**: Write access to the repository, security report routing, and dependency updating validation.

---

## 3. Decision-Making Process

All technical decisions should aim for technical consensus. If consensus cannot be reached, the Project Sponsor has final decision authority.

### Pull Request Reviews
- All PRs require review and approval from at least one Maintainer.
- Automated CI workflows (testing, security scans, SBOM generation) must pass successfully before a merge is permitted.
- A signed Contributor License Agreement (CLA) is mandatory for all external contributions.
