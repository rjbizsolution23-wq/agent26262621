# Security Policy

## Supported Versions

Only the latest release version of the Supreme AI Swarm Orchestrator (SASO) is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | ❌ No               |
| v2.0    |  Yes (Active)     |

---

## Reporting a Vulnerability

If you discover a potential security vulnerability, please do NOT create a public issue. Instead, report it directly to the Project Sponsor:

*   **Contact Email**: [rjbizsolution23@gmail.com](mailto:rjbizsolution23@gmail.com)
*   **Subject Prefix**: `[SECURITY VULNERABILITY] SASO`

We will acknowledge receipt within 48 hours and provide a detailed mitigation/remediation timeline.
